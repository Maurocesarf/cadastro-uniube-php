### Sistema de Cadastro de Produtos
#### Descrição
- Este é um sistema simples de cadastro de produtos desenvolvido em PHP. O projeto permite adicionar, listar, editar e excluir produtos de um banco de dados. Ideal para fins de aprendizado e como base para sistemas maiores.

##### Funcionalidades
- Cadastro de novos produtos
- Listagem de produtos cadastrados
- Edição de informações dos produtos
- Exclusão de produtos

##### Tecnologias Utilizadas
- PHP
- MySQL
- HTML
- CSS
